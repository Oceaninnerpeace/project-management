# project-management（Admin）

由 `create-rjgfny` 生成的管理端标准子项目。

## 能力

- `@rjgfny/ui-pro` Layout + 侧边菜单 + 标签页 + 路由 + Pinia
- `@rjgfny/ui-pro` + `@rjgfny/utils`
- `@rjgfny/config` 工程规范

## 开发

```bash
pnpm install
pnpm dev
```

## 工程规范

```bash
pnpm typecheck
pnpm lint
pnpm format
```

提交格式：`feat(scope): subject`
"# project-management"
